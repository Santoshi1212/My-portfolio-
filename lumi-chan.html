3D AI Assistant (Lumi-Chan)

Overview

Lumi-Chan is a smart, interactive 3D virtual assistant I’m developing entirely from scratch. It combines AI, natural language understanding, 3D avatar expressions, and personality to become a friendly and helpful assistant.

I developed Lumi-Chan and a pixel-style AI game using only a tablet and free tools — without access to a laptop or premium software. From AI to animations to UI design — I explored Python, JavaScript, HTML, CSS, and Godot — and built everything piece by piece. Imagine what I’ll do once I have access to professional-grade tools.

“If I can do all this with a tablet, I can do even more with the right opportunity."

Features

  • Chatbot with personality
  • Natural Language Understanding (NLU) using NLTK
  • Wikipedia-powered information search
  • Continuous conversation flow
  • VRoid-powered animated avatar (real-time facial + body animation in progress)
  • 3D UI using Three.js (switching to Godot for advanced control)
  • Smart memory and task management (in process)
  • Modular roadmap with AI upgrades planned

Tech Stack

  • Python, Flask, NLTK, Kivy, JavaScript, HTML/CSS, GitHub
  • Three.js, VRoid (real-time animation)
  • Acode and Dcoder (Android IDEs)
  • Godot (for advanced animation and functions)
  • Built entirely on Android - no PC or paid software
